Este projeto consiste em uma **Landing Page de alta conversão** desenvolvida para a empresa **Poder Financeiro**, com o objetivo central de captar leads qualificados e impulsionar o fechamento de contratos de **Antecipação do Saque Aniversário FGTS**.

Abaixo, os detalhes da descrição para o seu repositório:

### Sobre o Projeto
O sistema foi projetado para oferecer uma experiência **100% digital** e simplificada, permitindo que trabalhadores que optaram pelo Saque-Aniversário antecipem até **5 parcelas** do seu saldo. O foco principal é ajudar as famílias brasileiras a encontrarem um "respiro" financeiro e saírem do aperto através da liberação de crédito rápido e sem burocracia.

### Principais Funcionalidades e Metodologias
*   **Arquitetura de Persuasão:** O conteúdo foi estruturado com base no **Modelo AIDA** (Atenção, Interesse, Desejo e Ação) e na técnica dos **4 Us** (Útil, Urgente, Único e Ultraespecífico) para garantir headlines magnéticas e chamadas para ação (CTAs) poderosas.
*   **Design Responsivo:** Desenvolvido com **Bootstrap 5**, utilizando o sistema de **Grade (Grid)** de 12 colunas para garantir que a página seja visualmente impecável tanto em desktops quanto em smartphones (Mobile-first).
*   **Foco na Experiência do Usuário:** Interface moderna com suporte a **Bootstrap Icons**, componentes de **Cards** para exibição de benefícios e formulários otimizados com a classe `form-floating` para facilitar a captação de leads.
*   **Segurança e Autoridade:** Inclusão de seções de prova social e avisos de segurança ("Cuidado com Golpes") para construir confiança e autoridade perante o cliente.

### Tecnologias Utilizadas
*   **Bootstrap 5.1:** Framework principal para o layout e responsividade.
*   **HTML5 & CSS3:** Estrutura e estilização personalizada seguindo padrões modernos de web design.
*   **Bootstrap Icons:** Biblioteca de ícones para enriquecer a interface visual.
*   **Princípios de Copywriting:** Aplicação de gatilhos mentais de autoridade, reciprocidade e urgência para conversão imediata via WhatsApp.

### Estrutura Técnica
O projeto utiliza um template inicial integrado via **CDN** para garantir performance e redução no tempo de carregamento. A organização das pastas e do código segue as melhores práticas de desenvolvimento front-end, separando logicamente as seções de *Hero*, *Vantagens*, *Simulação* e *Rodapé*.
